Chapter 1. SigPML overview

The SigPML is a Domain Specific Modeling Language dedicated to data flow processing to create processing flows. The concrete syntax is based on blocks, ports and connectors which associated ports to create a directed flow. The execution semantics of the DSML must defined that all the blocks of a model run concurrently. Each block executes a local processing after a read operation on the input port. A write operation of the blocks on connectors through an output port is done after the local execution. The input and output ports own an admitted rate of data to transmit through the connectors. The connectors have a defined maximum date number. The input and output rates must respect the maximum data number.
