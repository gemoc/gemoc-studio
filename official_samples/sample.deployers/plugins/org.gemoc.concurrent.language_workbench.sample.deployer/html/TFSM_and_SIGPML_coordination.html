Overview

This example features the coordination of two GEMOC Concurrent languages : TFSM and SigPML.

It illustrates the GEMOC animation and coordination framework on two models, one SigPML model and one TFSM model.

Getting started

Install the projects for both TFSM and SigPML language:
File > new > Examples... > GEMOC TFSM Language example (Concurrent)
File > new > Examples... > GEMOC SigPML Language example (Concurrent)

Install the projects for the coordination of TFSM and SigPML language:
File > new > Examples... > GEMOC TFSM and SigPML coordination

Create and start an eclipse runtime:
Run > Run Configurations... > Eclipse application > new

Install sample models for the languages and their coordination:
File > new > Examples... > GEMOC model examples for TFSM and SigPML coordination
