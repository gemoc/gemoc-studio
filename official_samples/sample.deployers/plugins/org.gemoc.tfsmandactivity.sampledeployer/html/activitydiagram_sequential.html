Overview

This example features an executable language using GEMOC Sequential Engine.

It illustrates the GEMOC animation framework on an UML like Activity Diagram language.

More information and the sources of these projects are available on Github.

Getting started

Install the projects of this language:
File > new > Examples... >

Create and start an eclipse runtime:
Run > Run Configurations... > Eclipse application > new > ActivityDiagram Sequential Language

Install sample models for the language:
File > new > Examples... > ActivityDiagram Model examples (concurrent / sequential)

the model examples has been designed to work with both concurrent and sequential version of the language.
